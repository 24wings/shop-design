😰 栅格工具怎么用

Ant Design 设计系统全部基于 8pt 倍数设置栅格，library 中提供了常见的间距栅格，控制组件间的间距和对齐，形成规范美观的界面。使用后删除栅格即可。

如何在 Axure 里使用 Ant Design Library

1. 打开 Axure 软件，找到 Libraries 面板。

2. 点击  Libraries 面板右上角，三条杠图标 -> 点击“Load Library…”

3. 选择刚才在 library.ant.design 上下载的文件。

4. 导入成功，可直接在 Libraries 查看了。接下来只需将 Libraries 面板中的内容拖入画布使用。

常见疑问

😰 拖出组件太多，一个一个删太麻烦肿么办？

将组件拖入画布后，然后按住“shift”键，用鼠标点击要使用的组件，使该组件取消选择。再点击delete，即可删除剩余不需要使用的组件啦，一定要学会哟～

😰 icon显示乱码

首次使用icon需要查看icon组件下面的使用说明，按照说明安装icon字体和使用

安装时如遇到如下图状况，请勾选 iconfont.ttf 然后点击右下角安装选中的项目

如安装后，icon仍然显示乱码，重启axure即可。

😰 列表边线无法删除

模板中大部分的列表编辑都是通过 Style 面板中的 Border 属性，设置展示矩形的某一条或多条边线制作的，因此去掉或增加边框，需在此设置。

😰 组件太多找起来效率低

在 Libraries 面板中点击右上角搜索按钮，输入组件名称进行快速检索。清空搜索词后可展示全部。如果搜不到内容，请参考右图，切换分组。
